# DevVerse - Interactive Portfolio

A multi-page cosmic-themed portfolio built with vanilla HTML, CSS, and JavaScript.

## Development
1. Open with Live Server in VSCode
2. Or serve with local server: `python -m http.server 8000`
3. Navigate to `index.html`
